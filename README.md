# Emir Casil Cortez — Tattoo Artist Portfolio

A custom portfolio website created for my friend, tattoo artist **Emir Casil Cortez**. The site presents his artwork through an interactive sketchbook-inspired layout that feels personal, tactile, and connected to the tattoo process.

## About the Website

The portfolio is designed as a tilted paper sketchbook with a dark folder backing, textured surfaces, handwritten details, photo prints, and colored bookmark navigation. Visitors can browse Emir's introduction, artist profile, selected tattoo archive, and booking information without leaving the sketchbook experience.

## Features

- Responsive sketchbook and folder-inspired interface
- Internal bookmark navigation for each portfolio section
- Automatic and manually controlled featured-work slideshow
- Artist profile and studio details
- Editorial tattoo archive with real project photography
- Direct email booking call to action
- Responsive desktop and mobile layouts
- Reduced-motion support and accessible navigation labels

## Built With

- Next.js and React
- TypeScript
- Tailwind CSS
- Vinext and Vite
- OpenAI Sites / Cloudflare-compatible deployment output

## Run Locally

Requires Node.js `22.13.0` or newer.

```bash
npm install
npm run dev
```

Create a production build with:

```bash
npm run build
```

## Project Structure

- `app/page.tsx` — portfolio content, slideshow, and sketchbook navigation
- `app/globals.css` — visual system, paper textures, layouts, and responsive styling
- `public/` — production artwork, photographs, doodles, and social preview assets
- `IMGS/` — original source images supplied for the portfolio

## Purpose

This is a personal website project made for my friend and tattoo artist, Emir Casil Cortez, to give his work a dedicated online home and make it easier for potential clients to explore his style and start a project inquiry.

All tattoo artwork and photography belong to the artist and are used for this portfolio.
